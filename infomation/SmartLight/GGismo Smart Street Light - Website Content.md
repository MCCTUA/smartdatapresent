## ระบบจัดการไฟถนนอัจฉริยะเกินกว่าเพียงแค่โคมไฟ

Smart Street Light ไม่ใช่เพียงแค่การอัพเกรดโคมไฟถนน แต่เป็นการลงทุนในโครงสร้างพื้นฐานแบบอัจฉริยะที่จะเป็นรากฐานของเมืองอัจฉริยะในอนาคต ระบบนี้ผสานฮาร์ดแวร์ประสิทธิภาพสูง (155 lm/W) เข้ากับแพลตฟอร์มซอฟต์แวร์ที่ยืดหยุ่นและสามารถต่อยอดได้

<!-- Slide ID: slide_3_pain_points_1 -->
## ความท้าทายของการจัดการไฟถนนในปัจจุบัน

**หน่วยงานท้องถิ่นในประเทศไทยประสบปัญหาหลักในการจัดการระบบไฟฟ้าส่องสว่างสาธารณะ**

โคมไฟถนนแบบเดิมหรือ LED ธรรมดา ใช้พลังงานคงที่ตลอดคืน ไม่สามารถปรับลดความสว่างในช่วงเวลาที่มีการสัญจรน้อย (เช่น 00:00-05:00) ทำให้สิ้นเปลืองงบประมาณโดยไม่จำเป็น

เมื่อโคมไฟชำรุด หน่วยงานมักไม่ทราบจนกว่าจะมีการร้องเรียนจากประชาชน ส่งผลให้การแก้ไขปัญหาล่าช้า และใช้เวลาในการตรวจสอบและซ่อมแซมนาน

พื้นที่ที่มีไฟถนนชำรุดหรือไม่เพียงพอ เพิ่มความเสี่ยงต่อการเกิดอุบัติเหตุทางถนน การลักขโมย และส่งผลกระทบโดยตรงต่อคุณภาพชีวิตของประชาชนในพื้นที่

## GGismo Smart Street Lightโซลูชันครบวงจรสำหรับหน่วยงานท้องถิ่น

**ออกแบบมาเพื่อแก้ไขปัญหาทั้งหมด โดยการผสานฮาร์ดแวร์ประสิทธิภาพสูงเข้ากับแพลตฟอร์มซอฟต์แวร์ที่อัจฉริยะและยืดหยุ่น**








- โคมไฟ LED ประสิทธิภาพ 155 lm/W(สว่างกว่านวัตกรรมที่ขึ้นทะเบียน)
- ขาโคมไฟปรับองศาได้ 90°เพื่อปรับแสงให้เหมาะสมกับทุกสภาพพื้นที่

- ระบบควบคุมและตรวจสอบแบบ Real-timeผ่าน Web Dashboard และ Mobile App
- ระบบแจ้งเตือนอัตโนมัติเมื่อโคมไฟขัดข้อง
- ความสามารถในการปรับแต่งตามความต้องการของแต่ละท้องถิ่น

- ออกแบบแสงสว่างฟรีโดยทีมงานผู้เชี่ยวชาญ
- การสนับสนุนเทคนิค 24/7พร้อมดูแลและแก้ไขปัญหาตลอดเวลา

<!-- Slide ID: slide_6_how_it_works -->
## ระบบทำงานอย่างไร - สถาปัตยกรรมแบบ End-to-End

**ระบบ GGismo Smart Street Light ทำงานแบบ Integrated อย่างไร้รอยต่อ**

โคมไฟแต่ละดวงติดตั้ง Node LTE ที่สามารถตรวจวัดและส่งข้อมูล เช่น สถานะการทำงาน ระดับความสว่าง และปริมาณพลังงาน

ข้อมูลจากแต่ละโคมไฟถูกส่งไปยัง Cloud Server ผ่านเครือข่าย LTE/4G ทำให้สามารถติดตามได้ทุกที่ทุกเวลา

ระบบ Cloud ประมวลผลข้อมูลและเปรียบเทียบกับกฎเกณฑ์ที่ตั้งไว้ เพื่อตัดสินใจว่าควรทำการควบคุมอะไร

ระบบส่งคำสั่งไปยังโคมไฟ เช่น เปิด-ปิด หรือปรับระดับความสว่าง (Dimming) ตามตารางเวลาหรือตามความต้องการ

ข้อมูลทั้งหมดถูกบันทึกและแสดงผลใน Dashboard เพื่อให้เจ้าหน้าที่สามารถติดตามและวิเคราะห์ได้

<!-- Slide ID: slide_7_hardware_excellence -->
## ฮาร์ดแวร์ประสิทธิภาพสูง

**โคมไฟ GGismo ออกแบบด้วยเทคโนโลยี LED ล่าสุด เพื่อให้ได้ประสิทธิภาพสูงสุด ประหยัดพลังงาน สว่างกว่า และทนทาน**

![GGismo Street Light 90 Degree Adjustable](images/Gemini_Generated_Image_am4oo4am4oo4am4o.webp)

- สว่างกว่านวัตกรรมที่ขึ้นทะเบียน (125 lm/W) ถึง 24%
- ลดค่าไฟฟ้าได้ 20-30% เมื่อเทียบกับโคมไฟแบบเดิม

- ปรับมุมการส่องสว่างได้ตามสภาพพื้นที่
- เหมาะสำหรับถนนกว้าง ถนนแคบ และพื้นที่พิเศษ

- ทำงานได้ 24 ชั่วโมง/วัน เป็นเวลา 5+ ปี
- ผ่านการทดสอบมาตรฐาน IES LM-79 และ LM-80

- ป้องกันน้ำและฝุ่นได้อย่างสมบูรณ์
- ทนทานต่อสภาพแวดล้อมที่รุนแรง

## ระบบควบคุมอัจฉริยะ - ติดตั้งง่าย ใช้งานง่าย

**Node LTE ที่ติดตั้งในแต่ละโคมไฟ เป็นหัวใจของระบบควบคุม อ่านค่า ติดตามสถานะ และแจ้งเตือนอัตโนมัติ**

![Node LTE with Socket NEMA](images/RSzSPSegdeAuJPuP.png)

- ออกแบบให้ติดตั้งง่ายบนขาโคมไฟ ไม่ต้องเปลี่ยนโคมเดิม
- ใช้ Socket NEMA มาตรฐาน เข้ากันได้กับโคมไฟส่วนใหญ่

- Online / Offline Status
- Power (W)
- Current (A)
- Voltage (V)

- Lamp Failure (ไฟดับ)
- Power Failure (ไฟตัด)
- Over/Under Voltage
- Over Current
- Power Outage

- แสดงตำแหน่งโคมไฟทั้งหมดบน Dashboard
- นำทางไปยังโคมที่มีปัญหาด้วย Google Map ไม่ต้องขับรถตระเวน

**เพราะแต่ละท้องถิ่นมีบริบทที่แตกต่างกัน ระบบของเราจึงออกแบบมาให้ปรับแต่งได้ (Customizable)**

> **เป็นเจ้าของเทคโนโลยีและซอฟต์แวร์เองทั้งหมด**
>
> ทีมพัฒนาในประเทศไทยมีความเชี่ยวชาญด้าน Software ทำให้สามารถปรับปรุงและพัฒนาฟีเจอร์ใหม่ๆ ได้อย่างรวดเร็วและตรงจุด

*+ Tailored Reports ใน Premium Package*

### Local Support & Development
ทีมพัฒนาอยู่ในประเทศไทย พร้อมให้คำปรึกษา รับฟังปัญหา และปรับปรุงระบบอย่างรวดเร็ว ไม่ต้องรอการอัปเดตจากต่างประเทศ

## ผลประโยชน์และการคืนทุน - ลงทุนวันนี้ ประหยัดทุกวัน

**การลงทุนในระบบ Smart Street Light ไม่ใช่แค่รายจ่าย แต่คือการสร้างความคุ้มค่าในระยะยาว**

- ประหยัดพลังงานสูงสุด 50-70%ด้วยโคมไฟ LED ประสิทธิภาพสูง (155 lm/W) ผสานกับระบบหรี่ไฟอัตโนมัติ (Dimming) ในช่วงเวลาที่มีการสัญจรน้อย ช่วยลดค่าไฟฟ้ารายเดือนได้อย่างมหาศาล
- ลดค่าใช้จ่ายในการซ่อมบำรุงระบบแจ้งเตือนจุดที่ชำรุดทันที ไม่ต้องส่งเจ้าหน้าที่ขับรถตระเวนตรวจ (Truck Roll) ลดค่าน้ำมันและเวลาการทำงานของเจ้าหน้าที่
- จุดคุ้มทุนที่ชัดเจน (Break-even)สามารถนำงบประมาณค่าไฟฟ้าและค่าซ่อมบำรุงที่ประหยัดได้ในแต่ละปี มาชดเชยเป็นค่าใช้จ่ายในการลงทุนติดตั้งระบบ ทำให้คืนทุนได้ในเวลาไม่กี่ปี

- เพิ่มความปลอดภัยในชีวิตและทรัพย์สินแสงสว่างที่เพียงพอและสม่ำเสมอ ช่วยลดอุบัติเหตุบนท้องถนนและป้องกันปัญหาอาชญากรรมในพื้นที่เสี่ยง
- ยกระดับความพึงพอใจของประชาชนหน่วยงานสามารถรับรู้ปัญหาและเข้าแก้ไขโคมไฟชำรุดได้ทันทีก่อนที่ประชาชนจะร้องเรียนสร้างภาพลักษณ์ที่ดีในการบริหารงาน
- ก้าวสู่ Smart City และเป็นมิตรต่อสิ่งแวดล้อมลดการปล่อยก๊าซเรือนกระจก (Carbon Footprint) จากการประหยัดพลังงาน และวางรากฐานข้อมูลดิจิทัลเพื่อการพัฒนาเมืองในอนาคต

## ประสบการณ์ที่พิสูจน์แล้ว - คู่สัญญาของภาครัฐและเอกชนขนาดใหญ่

**โครงการสำเร็จที่สำคัญ พิสูจน์ความเชี่ยวชาญและความน่าเชื่อถือ**

ติดตั้งระบบไฟถนนอัจฉริยะ LED ประสิทธิภาพสูงให้กับเทศบาลและองค์การบริหารส่วนท้องถิ่นกว่า 50 แห่ง

พัฒนาระบบ Sensor ตรวจสอบและติดตามสภาพถนน สัญญาณไฟ และความปลอดภัยสำหรับทางด่วนทั้งหมด (ทุก Plaza)

ผลิตภัณฑ์ทั้งหมดได้รับการรับรองมาตรฐาน มอก. และผ่านการตรวจสอบคุณภาพอย่างเข้มงวด

พัฒนาระบบ AI/Machine Learning สำหรับติดตามพฤติกรรมผู้ต้องขับที่พ้นโทษ เพื่อความปลอดภัยของสังคม

สร้างแพลตฟอร์ม Enterprise ที่รองรับการติดตามข้อมูลจำนวนมากแบบ Real-time สำหรับหน่วยงานภาครัฐขนาดใหญ่

ทีมพัฒนามีความชำนาญในการพัฒนาระบบ IoT สำหรับงานโรงงานอุตสาหกรรม การวิเคราะห์ Data และ AI Analytics เพื่อการตัดสินใจของผู้บริหาร

- ระบบไฟถนน LED ทั่วประเทศติดตั้งระบบไฟถนนอัจฉริยะ LED ประสิทธิภาพสูงให้กับเทศบาลและองค์การบริหารส่วนท้องถิ่นกว่า 50 แห่ง
- ระบบตรวจสอบและติดตาม Sensor - ทางด่วนพัฒนาระบบ Sensor ตรวจสอบและติดตามสภาพถนน สัญญาณไฟ และความปลอดภัยสำหรับทางด่วนทั้งหมด (ทุก Plaza)
- มาตรฐาน มอก. และคุณภาพอย่างเข้มงวดผลิตภัณฑ์ทั้งหมดได้รับการรับรองมาตรฐาน มอก. และผ่านการตรวจสอบคุณภาพอย่างเข้มงวด

- ระบบ Machine Learning - กระทรวงยุติธรรมพัฒนาระบบ AI/Machine Learning สำหรับติดตามพฤติกรรมผู้ต้องขับที่พ้นโทษ เพื่อความปลอดภัยของสังคม
- ระบบ Dashboard & Monitoring ระดับองค์กรสร้างแพลตฟอร์ม Enterprise ที่รองรับการติดตามข้อมูลจำนวนมากแบบ Real-time สำหรับหน่วยงานภาครัฐขนาดใหญ่
- ทีม Dev เชี่ยวชาญด้าน IoT, Industrial, Data & AIทีมพัฒนามีความชำนาญในการพัฒนาระบบ IoT สำหรับงานโรงงานอุตสาหกรรม การวิเคราะห์ Data และ AI Analytics เพื่อการตัดสินใจของผู้บริหาร
